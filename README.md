# Hakerch Real Estate Fractional Investment Platform

## Overview
The Hakerch Real Estate Fractional Investment Platform is a mobile-first web application that allows users to buy, sell, and monitor fractional shares of real estate properties.  It makes real-estate investing more accessible by allowing users to invest as little as $1, track property performance, and view secure blockchain-verified ownership records.
